Clustr

Clustr is an open source data mining and visualization project designed to help local governments determine where resources are most needed.

The project uses data from the See Click Fix API and includes a data pipeline for processing and uploading the data, and a web app for visualizing the data.

The application features a heatmap, a map of clusters, and a street ranking for all open and acknowledged issues as of the last update.
